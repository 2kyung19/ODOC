## 문제 출처
- https://www.acmicpc.net/problem/2558

## 문제 풀이
- 간단한 입출력 문제, 이 정도 범주는 Scanner를 통해서 입출력 받아도 괜찮음
```java
import java.util.*;

public class Main{

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt();
        int b = sc.nextInt();
        System.out.println(a+b);
    }

}
```
