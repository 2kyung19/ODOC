## 문제 풀이

- dp[1] = 1 / dp[2] = 2 /dp[3] = 3 /dp[4] = 5 ...
- 쉽게 생각하면 새로 추가된 열이 2x1이냐 1x2이냐만 생각하면됨
- 새로 추가된 n열이 세로 줄인 경우, n-1까지 경우의 수와 동일
- 새로 추가된 n열이 가로 줄인 경우, 가로줄이면 n-1까지 침범하니까 n-2 경우의 수와 동일함
- 결국 총 경우의 수는 dp[n-1] + dp[n-2]
- 풀고나니 피보나치였던것,,

```python
n = int(input())

dp = [0]*(n+1)

dp[0]=dp[1]=1
for i in range(2,n+1):
    dp[i]=dp[i-1]+dp[i-2]

print(dp[n]%10007)
```
