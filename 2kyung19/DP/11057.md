## 문제 풀이

- 마지막에 오는 수의 경우의 수를 이중배열에 저장함
- i = i째 자리 / j = i째자리에 있는 숫자 / dp[i][j] = i째 자리에 있는 숫자의 경우의 수
- 1번째자리는 모두 1 > dp[1][0:9] = 1 (0 시작 가능)
- 0은 0에서만 올 수 있음 무조건 1이므로 for문 연산에서 제외함 range(1,10)
- 1~9는 0 ~ j 까지 모두 올 수 있음
- 0~j-1까지 계산된게 dp[i][j-1] + j에서 온 경우 dp[i-1][j]

```python
n = int(input())

dp = [[1]*(10) for _ in range(n+1)]

for i in range(2,n+1): # 첫자리 경우 1
    for j in range(1,10): # 0 = 1
        dp[i][j]=dp[i-1][j]+dp[i][j-1]

print(sum(dp[n])%10007)
```
